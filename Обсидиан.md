## Трюки
### Поиск внутри заметки
```
\```query
ваши поисковые критерии здесь
\```
```
Пример:
```query
Трюки
```

### Цитаты
Восклицательный знак перед ссылкой превращает ее цитату `![[ссылка]]`
- Пример: `![[A page name]]`
- Пример: `![[A page name#Section-Header]]`
- Пример: `![[A page name^Specific-Line from this note]]` (это даст вам любую строку **между 2 жесткими переносами строки**).

Живой пример:
